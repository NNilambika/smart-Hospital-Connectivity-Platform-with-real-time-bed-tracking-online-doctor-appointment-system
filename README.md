# Smart Hospital Connectivity Platform is a web-based system that provides real-time bed availability tracking and online doctor appointment booking.

## 📌 Project Description
Real time hospital management system that enables patients to check bed availability across hospitals instantly with color-coded status(green,yellow,red).patients can book doctor appointments online,receive queue
numbers via SMS,and access their health records including vitals and medical documenats. The platform provides hospitals with a compreshensive dashboard for bed management,doctor scheduling with present/absent 
toggle, patient vitals tracking, and document upload functionality. With intergrated emergency ambulance services featuring one-click calling and GPS location sharing,the system reduces bed finding time for hours 
to just 30 seconds.Built with HTML.CSS,JavaScript and Firebase,This solution connects patients,hospitals, and ambulances in one seamless platform.

## 🎯 Problem Statement
"In critical moments,patients and families waste precious hours searching for hospital beds while other stand in endless queues for doctor appointments-hospitals manage both manually, 
causing delays,stress, and inefficiency.There is no real time system connecting patients,beds, and doctors when it matters most".
